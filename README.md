# 🍷 Lumière

A modern, responsive fine dining restaurant website built with React and Vite.

> **Note:** This is the frontend only. No backend included.

---

## ✨ Features

- **Intro Loader** - Animated loading screen with logo reveal
- **Hero** - Full-screen background with CTAs
- **About** - Restaurant story and values
- **Features** - Why choose us
- **Menu** - Signature dishes with images
- **Gallery** - Clickable images with lightbox
- **Full Menu Modal** - Browse all categories
- **Testimonials** - Customer reviews
- **Responsive** - Works on mobile, tablet, laptop, and desktop

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| React | UI Framework |
| Vite | Build Tool |
| CSS | Styling |

---

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Then open http://localhost:5174
